### Projeto:
Projeto educacional em C++ para cálculo de média com exibição em tabela formatada, feito para o Trabalho 3 da disciplina de introdução a ciência da computação da Univali, professor **Lucas Debatins**.

### Grupo:
Gabriel Batista da Luz, Gustavo Vieira Jacintho, Henrique Luiz Gallassini, Hiago Ronaldo Gomes.
---

### Desafio proposto:

<p>Crie um programa em linguagem C ou C++ que permita cadastrar os
nomes e as três notas de até 5 alunos. Em seguida, o programa deve: </p> 	
<br>▪ Calcular a média das três notas para cada aluno.
<br>▪ Exibir uma tabela com nome, notas, média e situação (Aprovado
se média ≥ 7, Reprovado caso contrário).
<br>▪ Informar qual foi o aluno com a maior média da turma.
<br>▪ Use vetores para armazenar os dados dos alunos.
<br>▪ Use pelo menos uma função para calcular a média.
<br>▪ Use estrutura de repetição (for ou while) para entrada e exibição
de dados.
<br>▪ Utilize uma estrutura condicional (if/else) para determinar a
situação.

## Exemplo de Saída

```
------------------------------------------------------------------------------------------
Nome                                    Nota 1   Nota 2   Nota 3   Média     Situação
------------------------------------------------------------------------------------------
Henrique Gallasini                      2        1        3        2        Reprovado
Maria Souza                             9        8        10       9        Aprovado
------------------------------------------------------------------------------------------
```
